Danube

Formal methods for scientific computing, bridging data, numerics, and behavioral correctness in high-performance systems.

Project Overview

Danube studies how formal methods, numerical reasoning, and systems techniques can make scientific computing software more reliable while preserving performance on modern hardware.

The project focuses on the tension between data movement, numerical behavior, and correctness guarantees in high-performance computing.
